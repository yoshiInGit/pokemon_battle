## エントリーシーン

- ポケモンセット

payload : <ポケモン名>

- バトル開始
order : "start-battle"
payload : -

## バトルシーン
- 攻撃
order : "attack"

くさ　　：　良 -> みず、いわ  　　             悪 -> ほのお、ひこう、どらごん、はがね
エスパー：　良 -> かくとう  　　　             悪 -> はがね
でんき　：　良 -> みず、ひこう  　             悪 -> くさ、どらごん
どらごん：　良 -> どらごん　　　　             悪 -> はがね
はがね　：　良 -> いわ　　　　　　             悪 -> ほのお、みず、でんき
あく　　：　良 -> エスパー　　　　             悪 -> かくとう
いわ　　：　良 -> ほのお、ひこう　             悪 -> かくとう、はがね

ひこう　：　良 -> くさ、かくとう　　　　　　　　悪 -> でんき、いわ、はがね
